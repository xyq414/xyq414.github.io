# mcp-skill-probes

## 项目定位

mcp-skill-probes 是一组面向 MCP、Skill 和外部工具的真实可用性探针。它不追求一次性大而全，而是通过小测试确认工具边界、记录结果，并把经验沉淀成可复用的 agent 工作流。

## 关注问题

- 工具是否真的可调用
- 输入输出边界是否清晰
- 失败时属于权限、网络、格式还是工具能力问题
- 哪些能力适合进入长期工作流
- 哪些能力只适合保留为实验记录

## 当前状态

实验验证中。这个项目会持续记录工具能力变化和实际使用结论。
