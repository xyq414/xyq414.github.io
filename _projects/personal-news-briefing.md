# personal-news-briefing

## 项目定位 / Positioning

personal-news-briefing 不是单纯的爬虫，而是一个面向个人决策的信息流处理系统。它把多源新闻、财经与科技信息整理成可读、可复用、可长期迭代的每日 HTML 简报。

## 项目背景 / Background

碎片化信息源很容易制造噪音：新闻分散、财经信息更新快、科技趋势需要持续跟踪。这个项目的目标是把这些内容收束到一个稳定的个人工作流里，让每天的信息阅读从“到处找”变成“集中看、分层读、持续归档”。

## 核心能力 / Capabilities

- 多源内容整理
- 文本清洗与结构化
- LLM 摘要与分层标题
- 新闻、财经、科技板块扩展
- HTML 简报输出
- 长期个人工作流集成

## 当前成果 / Current Result

当前版本已经可以生成每日 HTML 简报，并形成固定的输出格式。公开 demo 页面已预留，安全脱敏后的样例会放在这里：[/projects/personal-news-briefing/demo/](/projects/personal-news-briefing/demo/)。

## 技术关键词 / Tech Stack

Python、HTML report、LLM summarization、automation workflow、information pipeline、agent-assisted development。

## 我的工作 / My Role

我负责需求设计、信息层级设计、摘要结构设计、工作流迭代、agent 协作开发和输出格式验收。这个项目体现的是人机协作工程实践：由人定义目标、判断质量和维护长期方向，由 agent 辅助完成代码实现、调试和结构化输出。

## 当前状态 / Status

成熟可用，持续迭代中。下一步计划包括科技板块扩展、公开版 demo、latest 页面、archive 历史归档，以及未来可能拆出的 public-core。
